HTML

Structure

  • html tag: contains all other HTML elements in the document
  • head tag: contains descriptive elements about the document
  • body tag: contains content to be displayed by the browser
  • meta tag: declares the character encoding is UTF-8
  • link tag: specifies external css file
  • script tag: specifies external JavaScript file

Elements

  • p tag: container for text and other HTML elements
  • div tag: container element used for logical grouping of content
  • a tag: anchor element used when adding links to a page

Special Character Entities

  • Can get special symbols by typing "&" followed by one of the following strings:
  • nbsp; nonbreakable space  
  • lt; less than symbol <
  • gt; greater than symbol >
  • copy; copyright symbol ©
  • trade; trademark symbol ™

List Elements

  • ul tag: unordered list
  • ol tag: ordered list
  • dl tag: description list
  • li tag: list item

Tables

  • table tag: defines a table
  • tr tag: defines a table row
  • th tag: defines a table header
  • td tag: defines table data/cell
  • caption tag: adds a caption to a table
  • To make a cell span more than one column use the attribute colspan
  • To make a cell span more than one row use the attribute rowspan

Semantic Structure Elements

  • header tag: contains the documents heading
  • main tag: contains the main unique content of the document
  • section tag: broad element for larger sections of content
  • article tag: used for blocks of content that can be read
  • aside tag: used for sidebars or grouping of nonessential elements like ads
